# Neural OS — Advanced AI Architect Platform

![Neural OS Header](https://via.placeholder.com/1200x400.png?text=NEURAL+OS+ARCHITECT+PLATFORM)

Neural OS is a premium, cinematic AI operating system and portfolio ecosystem designed for systems architects and AI engineers. It features a high-fidelity glassmorphism UI, real-time AI streaming via Groq, and a secure distributed architecture.

## 🚀 Features

- **iOS 26 Liquid Glass UI**: Ultra-premium design language with multi-layer blurs and ambient effects.
- **Groq-Powered AI Core**: Real-time architectural insights using Llama 3.3-70b.
- **Distributed Identity**: Secure authentication via Google and GitHub OAuth.
- **Architectural Workspace**: Managed deployments, real-time analytics, and portfolio orchestration.
- **Compliance Ready**: Integrated Terms of Service, Privacy Policy, and AI Governance frameworks.

## 🛠 Tech Stack

- **Frontend**: React, TypeScript, Vite, TailwindCSS, Framer Motion, Lucide React.
- **Backend**: Node.js, Express, Groq SDK, Nodemailer, Helmet.
- **Infrastructure**: Firebase (Auth, Firestore), Syncfusion (Enterprise Charts).

## 📦 Setup & Installation

### Prerequisites
- Node.js (v18+)
- Firebase Project
- Groq Cloud API Key

### Local Development

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/neural-os.git
   cd neural-os
   ```

2. **Configure Environment Variables**
   - Copy `.env.example` to `.env` in both root and `backend/` directories.
   - Populate with your credentials.

3. **Install Dependencies**
   ```bash
   # Root (Frontend)
   npm install

   # Backend
   cd backend
   npm install
   ```

4. **Run the Platform**
   ```bash
   # Terminal 1 (Frontend)
   npm run dev

   # Terminal 2 (Backend)
   cd backend
   node server.js
   ```

## 🛡 Security

- **Hardenened SSE**: Secure streaming with chunked transfer encoding.
- **DDoS Protection**: Rate limiting on sensitive AI and contact endpoints.
- **Vaulted Secrets**: Environment abstraction to prevent credential leaks.
- **CORS Whitelisting**: Strict production origin policies.
