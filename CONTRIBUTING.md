# Contributing to Neural OS

We are excited that you want to contribute to the Neural OS ecosystem!

## How to Contribute

1. **Fork the Repository**: Create your own copy of the project.
2. **Create a Branch**: Use a descriptive name for your feature or fix (e.g., `feat/neural-mesh`).
3. **Commit Your Changes**: Follow professional commit message standards.
4. **Push to GitHub**: Send your changes to your fork.
5. **Open a Pull Request**: Provide a detailed description of your contribution.

## Code Standards

- **TypeScript**: All new code must be fully typed.
- **Styling**: Adhere to the iOS 26 Liquid Glass design language.
- **Performance**: Ensure no significant impact on frame rates or bundle size.
- **Testing**: Include unit or integration tests where applicable.

## Architectural Review

All contributions will undergo a review by the core architects to ensure alignment with the platform's vision and security standards.
